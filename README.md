## üì¶ Instalasi (Cara Pasang)

Ikuti langkah-langkah ini perlahan untuk pemula:

### 1. Download Bot
Buka terminal (Command Prompt atau PowerShell) dan ketik perintah ini:

```bash
git clone https://github.com/sipaldrop/FightBot-Sipal.git
cd FightBot-Sipal
```

### 2. Install Aplikasi Pendukung
Masih di terminal, ketik perintah ini untuk menginstall semua kebutuhan bot:

```bash
npm install
```
*Tunggu sampai proses selesai. Mungkin butuh beberapa menit.*

## ‚öôÔ∏è Konfigurasi (Pengaturan)

### 1. Buat File Akun (`accounts.json`)
Kamu perlu membuat file untuk menyimpan data akunmu.
1.  Buat file baru bernama `accounts.json` di folder bot.
2.  Buka file tersebut dengan Notepad atau Text Editor.
3.  Isi data akunmu seperti format di bawah ini:

```json
[
  {
    "privateKey": "PRIVATE_KEY_METAMASK_MU",
    "proxy": "http://user:pass@ip:port"
  },
  {
    "privateKey": "PRIVATE_KEY_LAINNYA",
    "proxy": "http://user:pass@ip:port"
  }
]
```
> **Catatan:**
> - `privateKey`: Private key wallet EVM kamu (Hati-hati jangan disebar!).
> - `proxy`: (Opsional) Jika pakai proxy, isi formatnya. Jika tidak, hapus baris proxy.

### 2. Pengaturan Umum (`config.json`)
File `config.json` sudah diatur otomatis. Kamu bisa mengubahnya jika diperlukan:
```json
{
  "MINT_THRESHOLD": 2000,
  "SCHEDULED_HOUR": 7,
  "SCHEDULED_MINUTE": 30
}
```
